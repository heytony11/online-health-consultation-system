# Online Health Consultation System

## Project Description
Online Health Consultation System is a web application that allows patients to consult doctors online.

Patients can:
- Register
- Login
- View doctors
- Book appointments
- View appointment history

Doctors can:
- Register
- Manage profile
- View patient appointment requests

Admin can:
- Approve doctors
- View all appointments

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript
- Bootstrap

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Version Control:
- Git & GitHub

## Features

- Secure authentication
- Doctor listing
- Appointment booking
- Admin approval system
- Patient appointment history
- Doctor dashboard
